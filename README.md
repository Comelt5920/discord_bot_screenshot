# DiscordCap

A Python-based Discord bot that captures and sends screenshots to a designated channel.

## Features
- Captures screen on demand using a Discord command.
- Periodically captures and sends screenshots automatically.
- GUI for easy configuration of Bot Token, Channel ID, and intervals.

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install discord.py pyautogui
   ```
3. Run `Discord_Cap.py`.
4. Enter your Bot Token and Channel ID in the settings window.
5. Click **Start Bot**.

## Commands
- `!c` (default): Captures and sends a manual screenshot.
